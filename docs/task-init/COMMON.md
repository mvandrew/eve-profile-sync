# Общие требования к инструменту

Задача инструмента синхронизировать в выбранном профиле EveOnline настройки пользователей профиля и их чаров с выбранными настройками профиля и его чара в этом же профиле.

1. После запуска программа должна определить каталог с профилями EveOnline. Обычно - это каталог `C:\Users\{user}\AppData\Local\CCP\EVE\c_ccp_eve_online_tq_tranquility`.
2. Если каталог не найден, то предложить пользователю указать его.
3. Далее предложить выбрать профиль. Профили представляют собой имя каталога формата `settings_Crabs`, где `Crabs` - это имя профиля.
4. Далее предложить выбрать файл настроек пользователя, который выглядит `core_user_25237385.dat`, где `25237385` - ид пользователя. Если возможно, нужно показать не только ИД пользователя, но и его логин, если его можно выделить из dat файла.
5. Далее предложить выбрать файл настроек чара вида `core_char_2121857110.dat`, где `2121857110` - это ИД чара. Аналогично п.4 по возможности выделить имя чара. Если получиться, то ограничить выбор только чарами пользователя, который был выбран на 4 этапе.
6. Показать summary предстоящей подмены: выбранный профиль, пользователь и чар.
7. После подтверждения пользователем, сделать полную резервную копию каталога профилей в каталог текущей программы в подкаталог backup с именем вида: `settings_{Profile}_YYYYMMDD-HHmm.zip`.
8. В каталоге профиля выбранном на 3 этапе заменить содержимое файлов всех пользователей на файл выбранного пользователя, аналогично поступить с файлами настроек чаров.
9. Все выбранные параметры (профиль, пользователь, чар) сохранить в локальном файле настроек текущего проекта и при следующем запуске предлагать по умолчанию варианты выбранные на предыдущем этапе.

Целевая платформа Windows 10/11. 

Проект пишем на актуальной версии языка Go.

Инструмент полностью консольный. Все запросы в консоль, сообщения и пр. пишем только на английском языке.
